Команды Git
Команда Описание
git init Команда git init создает новый репозиторий Git. С ее помощью можно преобразовать существующий проект без управления версиями в репозиторий Git или инициализировать новый пустой репозиторий.
git clone url Клонирование репозитория Git с удаленного сервера на локальный компьютер, вместо url нужно указать ссылку.
git add file Команда, которая добавляет изменения в файлы в индекс Git. file - это имя файла.
git add . Команда, которая добавляет все измененные и новые файлы в текущем каталоге (и всех его подкаталогах) в индекс Git. Знак точки (".") в конце команды означает, что Git должен добавить все файлы в текущем каталоге и его подкаталогах.
git commit -m "comment" git commit - это команда, которая создает новый коммит в локальном репозитории Git. -m "comment" - это флаг, который позволяет добавить комментарий к новому коммиту.
git commit -a -m "comment" Флаг -a означает "автоматический" режим и позволяет добавить все измененные файлы в индекс перед созданием коммита.
git push url - это команда, которая отправляет изменения из локального репозитория Git на удаленный репозиторий. url
git status это команда, которая показывает текущее состояние вашего рабочего каталога и индекса в локальном репозитории Git.
git log команда, которая показывает историю коммитов в локальном репозитории Git.
git branch branch_name команда, которая используется для работы с ветками в Git. branch_name - это имя новой ветки, которую вы хотите создать.
git checkout branch_name -
git checkout -b branch_name -
git remote -
